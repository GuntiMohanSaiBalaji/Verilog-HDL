# Verilog Design & Simulation

Welcome to my personal repository dedicated exclusively to Verilog experiments, RTL design, and digital logic exploration.

## 🚀 Repository Objectives
* Learn and implement fundamental and advanced digital circuits.
* Practice hardware description language (HDL) coding standards.
* Simulate and verify module behavior using testbenches.

## 🛠️ Tools & Technologies Used
* **Language:** Verilog HDL
* **Simulation/Synthesis Tools:** Xilinx Vivado

## 📁 Repository Structure
* `/combinational/` - Adders, multiplexers, decoders, and ALUs.
* `/sequential/` - Flip-flops, registers, counters, and finite state machines (FSM).
